Line 61 Initiative
This initiative lead me to drafting a paper based on ARCgis pro simulations derived from using Metrolinx API and GTFS data. The goal was to train a machine learning algorithm for ideal stop placements and route scheduling, and with some postive conclusion about stop placements, and verhicly frequency and altertive line adjustments, MiWay's response resulted in a frequency increase to every 11 minutes in September 2023.
Backyard Door Design
This summer our 19 year old backyard door had finally given up against the summer heat, and provided an opportunity to design and construct a replacement door. Finalized a stronger frame and accounted for the mounting system for the heavier weight and dual-side accessible lock. Utilized Fusion360 and went to home depot, Yup door is in good shape first winter check.

MyWorkday Auto Submitter
A PyautoGUI program that automatically fills the tedious job application form of MyWorkday, using a chromium webdriver, it can bypass robot detection and preload responses that are configurable, such as availability and pulls the latest resume upload, cover letter and waits for submit request. Developed scraper to identify new opportunities using ScrapeOPS proxy and community spiders resulting in a summarized JSON file that can be parsed.
